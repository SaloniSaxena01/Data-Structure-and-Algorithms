# Strings

### Character Set
Character set is the set of characters whch are supported by some programming language.

#### ASCII Codes
Total 128 (0 to 127)
* A - Z : 65 - 90
* a - z : 97 - 122
* 0 - 9 : 48 - 57
* Enter : 10
* Spacebar : 13
* Escape : 27

ASCII code is stored in 1 byte in main memory.
